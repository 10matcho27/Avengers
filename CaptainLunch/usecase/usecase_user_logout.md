# ユースケース 6： 利用者がログアウトする

## 概要

利用者がホーム画面において，システムをログアウトする

## アクター

- 食堂の利用者

## 事前条件

- 利用者がシステムにログインしていること

## 事後条件

- 利用者がシステムからログアウトしていること
- システムがログイン画面を表示していること

## トリガ―

- 利用者がホーム画面でログアウトを押す

## 基本フロー

1. 利用者がメニュー画面で「ログアウト」ボタンを押す
2. システムは利用者をログアウト状態にし，ログイン画面を表示する
